
# SimpleQtChat

Простой чат на Qt C++.



## Установка

```bash
  https://github.com/versiyaV/SimpleQtChat.git
  cd SimpleQtChat
```
    
## Использование
Запуск сервера
```bash
cd Server
./server
```

Запуск клиента
```bash
cd client
./client
```
